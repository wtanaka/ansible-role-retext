wtanaka.retext
=============

Installs ReText

ReText is a simple but powerful editor for Markdown and
reStructuredText markup languages. ReText is written in Python
language and works on Linux and other POSIX-compatible platforms.

Example Playbook
----------------

    - hosts: servers
      roles:
         - wtanaka.retext
